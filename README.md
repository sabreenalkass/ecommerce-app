# 🛍️ E-Commerce React App

A modern and responsive e-commerce web application built with React and Tailwind CSS.

---

## 🚀 Live Demo

(https://ecommerce-8v3m7r386-sabreen-abualkas-projects.vercel.app)

---

## ⚙️ Tech Stack

- React.js
- Tailwind CSS
- React Router
- Context API
- LocalStorage

---

## ✨ Features

- 🛒 Add to cart system
- ❌ Remove from cart
- 💳 Checkout page (fake payment UI)
- 📦 Order history saved locally
- 🔐 Login / Register (fake authentication)
- 🎨 Modern responsive UI
- ⚡ Smooth animations
